# 🖤 FBP Command Center — Funky Black Poppy

**Bloom Dark. Sell Bold. Move Loud.**

Reselling command center for Funky Black Poppy. Three tools in one:

- **Command Center** (`index.html`) — Dashboard, Vendoo CSV upload, SS26 title refresh, daily queue
- **Listing Wizard** (`listing-wizard.html`) — 11-step AI-powered listing creator
- **Style Cards** (`style-cards.html`) — Social media product card generator

## Setup

1. Enable GitHub Pages: Settings → Pages → Source: main → Save
2. Your site will be live at: `https://funkyblackpoppy-art.github.io/`

## Usage

1. Open the Command Center
2. Drop your weekly Vendoo CSV export
3. Dashboard populates with KPIs, stale breakdown, platform coverage
4. Title Refresh tab shows all active listings with new SS26 titles
5. Use the Quick Title Generator or open the full Listing Wizard
6. Copy titles/tags/descriptions → paste into Vendoo
